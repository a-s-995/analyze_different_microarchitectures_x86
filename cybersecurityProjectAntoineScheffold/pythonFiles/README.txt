The file analyzeLogs_findSimilarities.py takes first a file with AMD instructions and second with Intel instructions as input, and creates three new files, one with the interception, one with just-Intel instructions, one with just AMD. 

The file justCmds.py takes as input a file generated by the sandsifter, and outputs a file, that my programm can use.

The file disasResultsWithX86dis.py takes a file with instructions as input and dissasembles them, using the X86dis dissassembler.
